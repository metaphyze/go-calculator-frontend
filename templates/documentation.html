Scientific Calculator

Scientific Calculator Functions

The following functions are available in the scientific calculator:

Functions List

  • abs(x): Returns the absolute value of x.
  • acos(x): Returns the arc cosine of x in radians.
  • acosh(x): Returns the inverse hyperbolic cosine of x.
  • asin(x): Returns the arc sine of x in radians.
  • asinh(x): Returns the inverse hyperbolic sine of x.
  • atan(x): Returns the arc tangent of x in radians.
  • atan2(y, x): Returns the arc tangent of y/x in radians.
  • atanh(x): Returns the inverse hyperbolic tangent of x.
  • cbrt(x): Returns the cube root of x.
  • ceil(x): Returns the smallest integer value greater than or equal to x.
  • copysign(x, y): Returns x with the sign of y.
  • cos(x): Returns the cosine of x in radians.
  • cosh(x): Returns the hyperbolic cosine of x.
  • dim(x, y): Returns the positive difference between x and y.
  • erf(x): Returns the error function of x.
  • erfc(x): Returns the complementary error function of x.
  • erfcinv(x): Returns the inverse of the complementary error function.
  • erfinv(x): Returns the inverse of the error function.
  • exp(x): Returns e raised to the power of x.
  • exp2(x): Returns 2 raised to the power of x.
  • expm1(x): Returns e raised to the power of x minus 1.
  • fma(x, y, z): Returns x * y + z accurately.
  • floor(x): Returns the largest integer value less than or equal to x.
  • gamma(x): Returns the gamma function of x.
  • hypot(x, y): Returns the square root of the sum of the squares of x and y.
  • j0(x): Returns the Bessel function of the first kind of order zero.
  • j1(x): Returns the Bessel function of the first kind of order one.
  • log(x): Returns the natural logarithm of x.
  • log10(x): Returns the base-10 logarithm of x.
  • log1p(x): Returns the natural logarithm of 1 + x.
  • log2(x): Returns the base-2 logarithm of x.
  • logb(x): Returns the binary logarithm of x (logarithm base 2).
  • max(x, y): Returns the larger of x and y.
  • min(x, y): Returns the smaller of x and y.
  • mod(x, y): Returns the remainder of x divided by y.
  • nan(): Returns a "not-a-number" value.
  • nextafter(x, y): Returns the next representable floating-point value after x towards y.
  • pow(x, y): Returns x raised to the power of y.
  • remainder(x, y): Returns the remainder of x divided by y, with the same sign as x.
  • round(x): Rounds x to the nearest integer.
  • roundtoeven(x): Rounds x to the nearest even integer.
  • sin(x): Returns the sine of x in radians.
  • sinh(x): Returns the hyperbolic sine of x.
  • sqrt(x): Returns the square root of x.
  • tan(x): Returns the tangent of x in radians.
  • tanh(x): Returns the hyperbolic tangent of x.
  • trunc(x): Truncates x to its integer part.
  • y0(x): Returns the Bessel function of the second kind of order zero.
  • y1(x): Returns the Bessel function of the second kind of order one.
